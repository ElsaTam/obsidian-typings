The following page lists the most important features and changes brought by each Electron version update.
A focus is put on the features that are most likely to be used by Obsidian developers.

New Electron versions can be installed by running the Obsidian installer again.

:::note
This page has been written from the perspective of a plugin developer, it is likely
that some features from the Chromium/Electron/Node.js updates are not listed here.

Links have been provided to the official changelogs where all changes can be found.
:::

:::danger
Users only sporadically update Obsidian's installer versions on desktop,
these features will only be available for most users after a couple of months.

Feel free to use these features in your personal plugins, themes and snippets.
:::
